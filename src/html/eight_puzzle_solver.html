8 Puzzle Solver in Go

The 8-puzzle is a smaller version of the slightly better known 15-puzzle. The 8-puzzle consists of an area divided into a 3 by 3 grid. On each grid square is a tile, expect for one square which remains empty. Thus, there are eight tiles A tile(numbered 1-8) that is next to the empty grid square can be moved into the empty space, leaving its previous position empty in turn. The aim of the 8-puzzle solver is to achieve a given goal configuration of tiles as follows(tile numbered 0 signifies empty space): This solver uses AI algorithms to solve random eight puzzle problems.

  • Breadth-First Search
  • Greedy Best-First Search
  • A* Search
